# Bookstore

A full-stack web application for managing a bookstore, built with Java Spring Boot backend and React TypeScript frontend.

## Architecture

```
bookstore/
├── backend/          # Spring Boot REST API
├── frontend/         # React + TypeScript UI
└── README.md
```

## Tech Stack

**Backend:**
- Java 21 + Spring Boot
- PostgreSQL (Docker)
- Spring Security + JWT
- Maven

**Frontend:**
- React 18 + TypeScript
- Tailwind CSS
- Axios
- React Router

## Features

- Book catalog management (CRUD)
- User authentication and authorization
- Shopping cart and order processing
- Admin dashboard
- Search and filtering

## API Documentation

When running, API documentation is available at `/swagger-ui.html`
